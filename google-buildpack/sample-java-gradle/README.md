Sample Java Gradle
------------------

Run Locally (with Java installed):
```
./gradlew run
```

Run Locally with Buildpacks & Docker:
```
pack build --builder=gcr.io/buildpacks/builder sample-java-gradle
docker run -it -ePORT=8080 -p8080:8080 sample-java-gradle
```
